# OkuBuilt Infrastructure & Cloud Portfolio

Welcome to my technical documentation portfolio.

This site documents infrastructure, cloud, automation, security, and operations projects. It is organized to show both hands-on infrastructure work and cloud implementations.

Local and on-premises systems administration work, including virtualization, storage, networking, containers, firewalls, monitoring, recovery procedures, and operational troubleshooting.

### Cloud

Cloud architecture and portfolio projects involving AWS, Azure, Terraform, CI/CD, Microsoft Entra, secure database design, serverless workflows, and cloud-hosted applications.
